PROJECT TITLE:
Battleship

PURPOSE OF PROJECT:
Game of Battleship.
Final project for Monterey Peninsula College (MPC) - CSIS 10A - Programming Methods I: Java class. 

DATE:
May 2019

HOW TO START THIS PROJECT:
Run GameRunner.java .

AUTHOR(S):
Junchen Wang

USER INSTRUCTIONS:
You will be playing against the computer.
Run GameRunner.java .
Place your ships.
The game will alternate between you and the computer targeting coordinates on the other's board.
If you have hit all of the computer's ships, you have won.
If the computer has hit all of your ships, you have lost.
